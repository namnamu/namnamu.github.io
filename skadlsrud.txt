java강의 ch1.

hello.java 소스코드가 jdk를 거쳐서 hello.class라는 기계어 코드가 생성된다.
그리고 실행됨
자바는 시스템이나 운영체제 없이 실행된다.
윈도우 c는 맥북, 리눅스 c로 100%호환이 안된다.
자바 가상머신이 있어서 어느 곳에서든 돌아갈 수 있는 것이다.
한줄씩읽어 실행시키는 과정을 인터프리터라고 한다.
자바는 인터프리터와 컴파일러를 모두 거치는 언어.
파이썬은 작성하면 인터프리터를 거쳐서 바로 결과가 보여짐. 
c는 인터프리터 과정없이 컴파일을 하면 바로 돌아감.

package 패키지명;//프로젝트 명이 아님 주의
public class 파일명(대문자로 시작){
    public static void main(String[] args)
    {
        System.out.println("Hello world");
    }
}

자바의 기본 형태
filename: Hello.java  -반드시 클래스 명이 파일 명이어야 함.
1.하나의 파일에 하나의 클래스를 작성하는 경우
import hellotest;// 프로젝트 명이 패키지 명이 된다.
class Hello{//클래스 명은 반드시 대분자로 시작.
//자바에서는 함수를 메소드라고 부른다.
  public static void main(String[] args){
     int data;
     float score;
     
     //if, for, while 구문 등
     System.out.println("~");//표준 출력문
  }
2.하나의 파일에 두개 이상의 클래스를 작성하는 경우
public class A{}
class B{}
A클래스명이 파일명이 되어야 한다., 하나의 클래스에만 붙일 수 있다.,public클래스가 없으면 파일명은 자유롭게 써도 된다., main 메소드는 하나의 클래스에만 존재해야 한다.

표기 규칙
변수명은 영문자(대소문자구별), _,$,숫자 사용 가능
클래스 이름의 첫글자는 항상 대문자.
패키지 이름은 항상 소문자
여러 단어로 이루어지는 경우 단어의 첫글자를 대문자로.
파이썬은 밑줄로 이루는 것을 권장하지만 자바는 대문자로 연결camalcase권장.
